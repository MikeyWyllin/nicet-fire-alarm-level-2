# NICET Fire Alarm Level II Practice Test

Static GitHub Pages quiz app.

## What is inside
- 300-question master bank
- 150 questions marked `Study Guide Derived`
- 150 questions marked `Generated Practice`
- 25 random questions per test
- 30-minute timer
- Auto-submit when time expires
- Local all-time stats saved in the browser
- Missed-question review mode

## Deploy
Upload every file in this folder to a GitHub Pages repo or branch. Keep `.nojekyll` in the root.

## Important
This is original study practice. Questions are pulled from official NICET practice tests and study guides.
